#1번의 시뮬레이션 과정

1. 변수초기화
2. N턴간 진행
  2-1.궁수의 턴
    2-1-1. 3명의 궁수가 적을 포착 (즉시 사살할 경우 같은적이 여러 궁수에게 공격당할 수 없다)
    2-1-2. 적을 사살
  2-2.적의 턴
  	2-2-1. 적이 한칸씩 내려옴
  	       -> 적의 array를 한칸씩 내리는 작업은 로드가 크다
  	       -> 궁수를 한칸씩 올리는 방식으로 변경. 궁수의 위치는 적을 포착할 때 값 변경으로 쉽게 가능하다.
3. 사살 수 확인
---------------------

매 시뮬레이션마다 사살수를 확인하여 최대값을 갱신하면 끝난다.




#실수한 부분
-------------------
String s = br.readLine();

M = s.charAt(2)-'0';
